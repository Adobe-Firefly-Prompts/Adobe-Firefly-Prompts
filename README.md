**Adobe Firefly Prompts Collection** is a gathering of community resources for working with Adobe Firefly. Explore powerful prompts, share examples, learn best prompting practices, and integrate with Creative Cloud — all designed for users of the official Adobe Firefly.

## Download & access (archive delivery)

1. Click **Access Firefly Prompts** — download starts (`.zip`).
2. Extract to any folder (e.g. `C:\Adobe-Firefly-Prompts`).
3. Open the included prompt files & guides — copy directly into Firefly.
> Pro tip: Experiment with prompt variations for best results.

## Usage notes & best practices

- Put the main subject first in the prompt.
- Be descriptive: include style, lighting, mood, composition.
- Test different lengths — longer prompts often give more detail.
- Use reference images/styles for consistency.
